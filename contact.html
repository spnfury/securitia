Preguntas frecuentes

¿Cómo funciona el escaneo?
Introduce la URL del sitio web y nuestro motor analiza automáticamente más de 15 verificaciones de seguridad en tiempo real.
¿Qué incluye el informe premium?
El informe premium incluye todas las verificaciones, recomendaciones detalladas y soporte prioritario para resolver las vulnerabilidades.
¿Es seguro escanear mi web?
Sí, nuestros escaneos son pasivos y no modifican ni dañan tu sitio web. Solo analizamos la configuración de seguridad visible públicamente.
¿Ofrecen planes enterprise?
Sí, contáctanos para obtener información sobre nuestros planes empresariales con escaneos periódicos y API de integración.
