Términos de Uso

Condiciones de uso del servicio Securitia

1. Aceptación de los términos

Al acceder y utilizar el servicio Securitia, aceptas estar sujeto a estos Términos de Uso. Si no estás de acuerdo con alguna parte de estos términos, no debes utilizar nuestro servicio.

2. Descripción del servicio

Securitia es una plataforma de análisis de vulnerabilidades web que permite escanear sitios web públicos para detectar posibles problemas de seguridad. El servicio incluye:

  • Análisis de configuraciones de seguridad HTTPS/TLS
  • Verificación de cabeceras de seguridad
  • Detección de archivos sensibles expuestos
  • Informes con recomendaciones de corrección

3. Uso permitido

El servicio debe utilizarse únicamente para:

  • Analizar sitios web de tu propiedad o sitios para los que tengas autorización explícita.
  • Propósitos de seguridad legítimos y éticos.
  • Mejorar la postura de seguridad de los sitios analizados.

4. Uso prohibido

Está estrictamente prohibido:

  • Utilizar el servicio para escanear sitios sin autorización del propietario.
  • Intentar explotar vulnerabilidades encontradas para obtener acceso no autorizado.
  • Utilizar el servicio con fines maliciosos, de espionaje o competencia desleal.
  • Realizar escaneos automatizados masivos sin autorización previa.
  • Vender o redistribuir los informes sin autorización.

5. Renuncia de garantía

Los resultados proporcionados por Securitia son de carácter informativo y no constituyen una auditoría de seguridad profesional completa. El servicio se proporciona "tal cual" sin garantías de ningún tipo, expresas o implícitas.

No garantizamos que:

  • El servicio detecte todas las vulnerabilidades existentes.
  • Los resultados sean completamente precisos o estén actualizados.
  • El servicio esté libre de interrupciones o errores.
